WHAT IS APACHE ?

Apache Web Server is a free and open-source web server software that is widely used to host websites on the internet. It was developed by the Apache Software Foundation and is available for a variety of operating systems including Linux, Unix, and Windows. Apache is known for its high performance, reliability, and security features, and it is one of the most popular web server software in use today. Apache supports multiple programming languages, such as PHP, Perl, and Python, and it can be extended through the use of modules to provide additional functionality.

WHAT IS LIVE SERVER?

This is a little development server with live reload capability. Use it for hacking your HTML/JavaScript/CSS files, but not for deploying the final site. There are two reasons for using this: AJAX requests don't work with the file:// protocol due to security restrictions, i.e. you need a server if your site fetches content through JavaScript. Having the page reload automatically after changes to files can accelerate development.

    Flex Box

    Flexbox is a layout model that allows you to create flexible and responsive layouts for your web pages.
    It makes it easier to align and distribute items within a container, regardless of their size or shape.

    The main properties of Flexbox are:

    - display:

    This property is set to flex to create a flex container.

    - flex-direction:

    This property specifies the direction of the main axis. It can be set to row for a horizontal layout or column for a vertical layout.

    - justify-content:

    This property aligns items along the main axis. It can be set to ,flex-start, center, flex-end, space-between, or space-around.

    - align-items:

    This property aligns items along the cross axis. It can be set to flex-start, center, flex-end, stretch, or baseline.

    - flex-wrap:

    This property specifies whether the items should wrap when they exceed the width of the container. It can be set to nowrap, wrap, or wrap-reverse.

    - flex-grow:

    This property specifies how much an item should grow to fill the available space.

    - flex-shrink:

    This property specifies how much an item should shrink when there is not enough space.

    -flex-basis:

    This property specifies the initial size of an item before it grows or shrinks.

    Positioning in css

    CSS positioning is a way to control the layout of HTML elements on a web page.

    There are several types of positioning in CSS:

    1.

    Static positioning:

    This is the default positioning of an element. It follows the normal flow of the document and cannot be moved.

    2.

    Relative positioning:

    This type of positioning moves an element relative to its current position.
    It allows you to adjust an element's position without affecting the layout of other elements on the page.

    3.

    Absolute positioning:

    This type of positioning removes an element from the normal flow of the document and positions it relative to the nearest positioned ancestor. If there is no positioned ancestor, it positions relative to the body element.

    Fixed positioning:

    This type of positioning positions an element relative to the viewport, meaning it stays in the same place even if the page is scrolled.

    Sticky positioning:

    This type of positioning is a hybrid of relative and fixed positioning. It allows an element to be positioned relative to its normal position until it reaches a certain point, at which point it becomes fixed.

    Here's an example of CSS code for absolute positioning:

    ```css .element { position: absolute; top: 50px; left: 100px; } ```

    This would position the element 50 pixels from the top and 100 pixels from the left of its nearest positioned ancestor.

    CSS BOX MODEL

    The CSS box model is a concept that describes how elements in HTML are structured and how their content, padding, border, and margins are positioned and sized. It is important to understand this concept when designing layouts with CSS.

    Diagram of Css Box Model :

    The box model can be visualized as a rectangular box that surrounds an element's content and defines its dimensions.

    The four main components of the box model are:

    1.

    Content:

    This is the actual content of the element, such as text, images, or other HTML elements.

    2.

    Padding:

    This is the space between the content and the element's border.
    It can be set using the CSS padding property.

    3.

    Border:

    This is the line that surrounds the element's padding.
    It can be set using the CSS border property.

    4.

    Margin:

    This is the space between the element's border and the neighboring elements. It can be set using the CSS margin property.

    CSS code example:

    ``` .box { width: 200px; /* sets the width of the box */ height: 100px; /* sets the height of the box */ padding: 20px; /* adds 20 pixels of padding around the content */ border: 1px solid black; /* adds a 1 pixel border around the padding */ margin: 10px; /* adds 10 pixels of margin around the border */ } ```
